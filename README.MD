# AWS Projects – Cloud Operations Focus

This repository contains **hands-on AWS projects** with a focus on:
- Operations
- Supportability
- Monitoring
- Troubleshooting

These are not demo-only projects. Each one reflects real AWS usage patterns.

## What This Repo Shows
- Comfort navigating AWS services
- Understanding how services interact
- Awareness of operational risks and failure points

## Key AWS Services Used
- EC2
- IAM
- CloudWatch
- VPC
- S3
- AWS CLI

## Operational Skills Demonstrated
- Service configuration and validation
- Permission troubleshooting
- Monitoring setup and log inspection
- Identifying misconfigurations that cause outages

## How These Projects Are Structured
Each project includes:
- What was built
- Why it was built
- Common failure scenarios
- How to verify and troubleshoot

## Who This Is For
- Cloud Operations Engineers
- Infrastructure Support Engineers
- AWS Support roles
- NOC / SRE-adjacent teams

If you’re reviewing this repo as part of an interview process, focus on how issues are **identified and resolved**, not just deployed.
